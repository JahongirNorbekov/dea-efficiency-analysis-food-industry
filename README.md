# dea-efficiency-analysis-food-industry
Applied Data Envelopment Analysis (DEA) to assess efficiency of food industry companies, identifying benchmarks and performance gaps.
## Key Insights
- Conagra, General Mills, and Campbell are fully efficient (score = 1.000)
- Mondelez and Hormel show moderate inefficiency
- Tyson is highly inefficient and requires significant input reduction

## Conclusion
The analysis highlights performance differences across firms and identifies benchmark companies for improving operational efficiency.
